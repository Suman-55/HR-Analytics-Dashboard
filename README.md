# HR Analytics Dashboard

## Overview

This project analyzes employee data using Power BI and SQL to uncover workforce trends, employee attrition patterns, salary insights, and department performance.

The dashboard helps HR teams make data-driven decisions to improve employee retention and workforce planning.

---

## Business Problem

The HR department wants answers to the following questions:

- Why are employees leaving?
- Which department has the highest attrition?
- Which age group leaves the most?
- Does overtime increase attrition?
- Which job role has the highest turnover?
